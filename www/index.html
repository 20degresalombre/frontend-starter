How to use it ?

As simple as this:

  • Clone the Github repo 
    git clone git@github.com:20degresalombre/frontend-starter.git
  • Install node modules 
    npm install
  • To launch a local webserver, compile your sources (.scss, .js, .tpl.html files), watch, recompile and refresh the view when you save a file just do : 
    gulp
  • Go to http://localhost:4000/
  • Modify/add HTML, SCSS, JS files and let the magic happen :D
